# Changelog

Todos los cambios notables de **K8Dash** se documentan en este fichero.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/)
y este proyecto sigue [Versionado Semántico](https://semver.org/lang/es/).

Cada entrada agrupa los cambios incluidos en una versión (imagen
`fermdez96/k8dash:<versión>`), enumerando los commits mergeados desde `master`
hacia la rama de publicación desde la release anterior.
